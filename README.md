# Appenheimer

A highly interactive, consumer-facing software discovery platform.

## Philosophy

Appenheimer helps users discover software and websites using deterministic search techniques. It focuses on a highly interactive, lightning-fast search experience. It is not an AI product, not a SaaS dashboard, not a CRUD application, and not a blog. Production will never depend on AI. The repository is structured for long-term maintainability and open-source development.
