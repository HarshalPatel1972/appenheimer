# Appenheimer Service Level Objectives (SLOs)

These metrics represent the operational contract for the platform.

| Metric | Target |
|---|---|
| Search P95 | < 150 ms |
| Search P99 | < 300 ms |
| Publish Latency | < 2 s |
| Worker Backlog | < 100 pending events |
| Error Rate | < 0.1% |
| Availability | 99.9% |
