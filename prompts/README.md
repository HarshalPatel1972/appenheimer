# Prompts Index

This directory stores prompts designed for interacting with AI coding assistants and automation tools.

## Directories and Usage

- **gent/**
  - **Purpose:** Prompts that define the overarching behavior, persona, or workflow for autonomous agents.
  - **When to use:** When initializing an agent session or providing it with long-term context and operational rules.

- **eview/**
  - **Purpose:** Prompts used for code review, security audits, and architectural validation.
  - **When to use:** Before merging a PR, or when requesting an AI assistant to critique a specific implementation against project standards.

- **planning/**
  - **Purpose:** Prompts for generating implementation plans, breaking down complex tasks, and exploring architectural options.
  - **When to use:** When starting a new feature or migrating a system, before writing any code.

- **efactor/**
  - **Purpose:** Prompts detailing how to modernize, clean up, or reorganize existing code.
  - **When to use:** When reducing technical debt or aligning legacy code with new architectural standards.
